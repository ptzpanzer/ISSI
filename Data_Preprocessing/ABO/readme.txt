1. Run script 'main.py' in ABO_download to download the NOAA ABO dataset. 
	- The script is already annonymized. Although the dataset is publicly avaliable, you first need to contact https://madis.ncep.noaa.gov/data_application.shtml to get your own access.

2. Run script 'gz_processing.py' in ABO_Process to preprocess the NOAA ABO dataset to the standard format.
